flex

기본 축을 기준으로 요소들을 정렬 

        * 부모요소 : flex-container
            - display : flex; => flex를 사용하겠다는 알림
            - flex-direction : row(기본값) | column | row-reverse | column-reverse ;
            - justify-content : 메인축 정렬(flex-direction이 row)
            - align-items : 크로스축 정렬(flex-direction이 row라면 column, column이라면 row)

            - flew-wrap : 줄바꿈여부
            - align-content : 여러줄인 경우의 크로스 축 설정

        * 자식요소(정렬의 주체) : flex-item
            - flex-grow : 빈 여백을 아이템들의 주어진 배율대로 늘어나면서 레이아웃을 채움
            - flex-shrink : 넘친 아이템 영역을 분배하여 레이아웃 영역 안에 아이템들이 배치되도록 하는 
            - flex-basis : 아이템의 크기를 직접 지정함
