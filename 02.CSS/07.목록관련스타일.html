목록 관련 스타일(list~ )

list-style-type : 불릿기호를 변경시킬 때 사용

list-style-image : 블릿기호로 이미지를 적용시킬 때 사용

list-style-position : 블릿기호의 위치를 조정할 때 사용

        * 순서 없는 목록 (ul)
        선택자 {
            list-style-type : disc|circle|square|none;
        }


        * 순서 있는 목록 (ol)
        선택자 {
            list-style-type : decimal|decimal-leading-zero;
            list-style-type : lower-alpha|upper-alpha;
            list-style-type : lower-roman|upper-roman;
        }

        * 위치 조정
        선잭자{
            list-style-position : outside(기본값)|inside;
        }
